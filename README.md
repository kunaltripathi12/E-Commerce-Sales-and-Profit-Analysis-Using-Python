- E-Commerce-Sales-and-Profit-Analysis-Using-Python
End-to-End Data Analysis Project using Python to analyze E-commerce Sales and Profit Trends.

This project analyzes an E-commerce dataset to understand sales trends, profit performance, customer segments, and product category performance. The goal of this project is to extract meaningful business insights and provide recommendations for improving profitability and sales strategies.

The analysis was performed using Python libraries such as Pandas, plotly in Jupyter Notebook.

Project Objectives

- Analyze monthly sales trends
- Identify the most profitable product categories
- Understand profit contribution by sub-category
- Analyze customer segments
- Evaluate sales-to-profit efficiency
